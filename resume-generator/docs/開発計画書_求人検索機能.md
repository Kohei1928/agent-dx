# より転-DX 求人検索機能 開発計画書

**作成日**: 2026年1月5日  
**最終更新**: 2026年1月5日  
**ステータス**: Phase 1完了

---

## 1. 概要

### 1.1 目的
CAがcircusのように求人を検索し、求職者に提案できるようにする

### 1.2 対象ユーザー
- **CA**: 求人検索、提案シート作成
- **RA**: 求人登録・管理
- **管理者**: 全機能

---

## 2. 機能一覧

| # | 機能 | 優先度 | 担当ロール |
|---|------|--------|-----------|
| 1 | 求人検索（CIRCUS風） | P0 | CA |
| 2 | 求人登録・編集 | P0 | RA, 管理者 |
| 3 | 選考一覧（CIRCUS風） | P0 | 全員 |
| 4 | 選考ステータス拡張 | P0 | CA |
| 5 | 面接詳細管理 | P1 | CA |
| 6 | 提案シート拡張 | P1 | CA |
| 7 | 求人レポート | P2 | 全員 |

## 4. データモデル

### 4.1 選考ステータス（拡張）

```prisma
enum SelectionStatus {
  // 提案フェーズ
  proposal              // 提案中
  not_applying          // 応募しない
  
  // 書類フェーズ
  document_submitted    // 書類提出済み
  document_screening    // 書類選考中
  document_passed       // 書類通過
  document_rejected     // 書類不通過
  
  // 面接フェーズ
  interview_scheduling  // 面接調整中
  interview_confirmed   // 面接確定
  first_interview       // 一次面接
  second_interview      // 二次面接
  final_interview       // 最終面接
  
  // 結果フェーズ
  offer                 // 内定
  offer_accepted        // 内定承諾
  offer_rejected        // 内定辞退
  rejected              // 不採用
}
```

### 4.2 面接詳細

```prisma
model InterviewDetail {
  id              String    @id @default(cuid())
  selectionId     String
  interviewRound  Int       @default(1)
  
  // 日程（日程調整から自動入力）
  scheduledAt     DateTime?
  duration        Int?      // 分
  
  // 形式
  format          InterviewFormat @default(online)
  location        String?   // 対面の場合
  onlineUrl       String?   // オンラインの場合
  
  // 詳細
  interviewers    String?   // 面接官
  preparation     String?   // 準備事項
  dressCode       String?   // 服装
  notes           String?   // 注意事項
  
  selection       Selection @relation(...)
}

enum InterviewFormat {
  online
  onsite
}
```

## 5. API設計

| エンドポイント | メソッド | 説明 |
|---------------|---------|------|
